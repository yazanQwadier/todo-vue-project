# Todo Tasks Project
### - Front End
 - Vue.js
 - Bootstrap
 - sweetalert2

### - Back End
- Pure php 
- Phinx
- PhpUnit

---

## Features :
- (Add/Update/Remove) Tasks
- Categorize Your Tasks By Time (daily/weekly/monthly/yearly)
- Mark Some Tasks As Done
